# 🎨 Gradient Generator

A simple interactive tool to generate background gradients with live preview.

## 🚀 Features
- Random gradient color generation  
- Direction control (left, right, top, diagonal)  
- Live preview  
- One-click copy of CSS code  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript
